🎨 PXS v3.0 - Consumer API Examples

This page demonstrates the consumer-facing API - how to use PXS in your applications.

Example 1: High-Level API - Load Image

The simplest way to use PXS - one method call:

// ✨ ONE LINE - Load image and render
const frameData = await animator.loadImage('photo.jpg', {
  quality: 'high'
});
// Frame data is automatically rendered and available

Example 2: Get Data - Access Current State

Get the current grid as PXSFrame data:

// Get current state
const frameData = animator.getData();
console.log(frameData.cells.length); // Number of cells
console.log(frameData.cells[0]); // { x: 0, y: 0, color: 'rgb(...)' } 

  


  

    
Example 3: Edit Data - Modify Pixels

    
Edit the data structure, then render it back:

    

      
// Get data

const frame = animator.getData();



// Edit specific cells

frame.cells[100].color = '#FF0000';  // Red

frame.cells[101].color = '#00FF00';  // Green

frame.cells[102].color = '#0000FF';  // Blue



// Render changes

await animator.setData(frame);
      
    

    
  


  

    
Example 4: Export/Import - Save & Load

    
Export as JSON, save anywhere, import later:

    

      
// Export

const json = animator.exportData({ pretty: true });

localStorage.setItem('my-art', json);



// Import

const saved = localStorage.getItem('my-art');

await animator.importData(saved);
      
    

    
    
    

  


  

    
Example 5: Storage Adapters - Save Anywhere

    
Use different storage backends:

    

      
// LocalStorage (fast, ~5MB)

await PXSStorage.local.save('my-art', frameData);

const art = await PXSStorage.local.load('my-art');



// IndexedDB (large storage)

await PXSStorage.indexedDB.save('big-animation', animData);
      
    

    
    
  


  

    
Example 6: Pattern Helpers - Generate Gradients

    
Use helpers to create patterns:

    

      
// Generate gradient

const gradient = PatternHelpers.generateRadialGradient({

  centerX: 20, centerY: 15,

  colorCenter: '#FFD700',

  colorEdge: '#1a1a2e',

  gridWidth: 40, gridHeight: 30

});



// Apply to animator

animator.updateCells(gradient);
