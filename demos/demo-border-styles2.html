📖 Border Style API

Control border appearance with CSS border-style property:

const animator = new CellAnimator({
    cellBorders: true,
    borderColor: '#333333',
    borderWidth: 1,  // Default: 1px
    borderStyle: 'inset'  // NEW! CSS border-style
});

Per-cell override example:

animator.updateCell(5, 10, {
    cellBorders: true,
    borderColor: '#ff00ff',
    borderWidth: 3,
    borderStyle: 'ridge'  // Per-cell override
});
⚠️ Canvas Mode Limitation: Canvas 2D context only supports solid lines. The borderStyle property is accepted for API consistency, but Canvas will always render solid borders. Use HTML mode for full border-style support.
