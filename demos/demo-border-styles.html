📖 Configuration API

Customize cell borders with these new options:

const animator = new CellAnimator({
    width: '100%',
    height: '100%',
    cellWidth: 20,
    cellHeight: 20,
    container: document.getElementById('container'),
    renderMode: 'canvas',  // or 'html'
    
    // Border configuration
    cellBorders: true,           // Enable/disable borders (default: true)
    borderColor: '#00ffff',      // Border color (default: '#333333')
    borderWidth: 2               // Border width in pixels (default: 1)
});

Examples:

  • cellBorders: false - No borders at all
  • borderColor: '#ff00ff' - Magenta borders
  • borderWidth: 3 - Thicker 3px borders
  • borderColor: 'rgba(255, 255, 255, 0.1)' - Semi-transparent white (Canvas only)
